# Anurag Kumar Cyberpunk Portfolio

Futuristic MERN portfolio for a tech researcher and cinematic content creator.

## Stack

- Frontend: React, Vite, Tailwind CSS, Framer Motion, Three.js
- Backend: Node.js, Express, MongoDB, JWT, Multer, Cloudinary

## Run

1. Install dependencies:
   `npm install`
2. Create env files from:
   - `client/.env.example`
   - `server/.env.example`
3. Start both apps:
   `npm run dev`

## Deploy

- Client: Vercel
- Server: Render
- Database: MongoDB Atlas
